# Refocus Frontend

# Required items

- Nodejs v18.20.3
- Backend must be running
- Change NEXT_PUBLIC_API_URL path in file `.env.local`

# Command

- Local setup
`npm install && npm run dev`

- Access url on local `http://localhost:3000`

# Site Flow

- Click on Login Page
- Enter username `admin` and password `admin`
- Will take you to dashboard
- Add new user
- Edit user
- Delete user
